# Design Tokens

Single source of truth for colors, spacing, radius and typography, pulled from the
**KNKY - DS** Figma file's Variables. Defined in `src/tokens/tokens.css` as CSS custom
properties (`--knky-*`). Reuse these in every new component instead of hardcoding raw
values — the swatches below always show the live value currently in `tokens.css`.
